## ⬇️ Download

Get the latest **musify.exe** from the [Releases](https://github.com/Akunimal/Musify-for-Windows/releases) page.  
No installation required — just download and run.

---

## Features

- Online song search with suggestions
- Offline listening support
- Import & export your data
- High-quality audio streaming
- Beautiful, responsive UI
- **Native Windows x64 build** (no emulation)

---

## Building from source

```bash
flutter pub get
flutter build windows --release
```

The executable will be at `build\windows\x64\runner\Release\musify.exe`.
